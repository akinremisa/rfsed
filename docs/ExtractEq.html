Module ExtractEq

rfsed.ExtractEq.ExtractEq(datapath, filename, catalog, stalat, stalon, Request_window)

Extract the data for each earthquake from the local data files Writes the extracted data to a new file

Parameters:

datapath – Path to the local data files directory

:type datapath:str :param filename: Path to the new file :type filename:str :param catalog: Path to the catalog file :type catalog:str :param stalat: Latitude of the station :type stalat:float :param stalon: Longitude of the station :type stalon:float :param Request_window: Time window relative to first P arrival :type Request_window:list[start, end] :return: Saves extracted earthquake data into a new file
